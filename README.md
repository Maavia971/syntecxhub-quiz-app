# Syntecxhub@-Quiz-App

A modern, interactive **frontend quiz application** built with **HTML, CSS, and JavaScript**. Designed for learning and testing knowledge across multiple categories with instant feedback and confetti celebration for correct answers.  

---

## 📝 Project Description

This is a professional-grade quiz app project suitable for internships or portfolio use. It allows users to:

- Select from multiple quiz categories
- Answer multiple-choice questions
- View real-time progress and score
- Enjoy a confetti animation for successful scores
- Track performance with instant feedback messages

The app is fully responsive, lightweight, and designed to give a clean and engaging user experience.

---

## 🚀 Features

- **Category Selection** – Choose different quiz categories
- **Dynamic Questions** – Questions load dynamically from JavaScript objects
- **Score Tracking** – Instant score updates after each question
- **Progress Bar** – Visual progress indicator for quiz completion
- **Confetti Animation** – Celebratory confetti for scores greater than 0
- **Responsive Design** – Works seamlessly on mobile and desktop
- **Light/Dark Theme Toggle** – User-friendly theme option

---

## 🛠️ Technologies Used

- **HTML** – Markup for structure
- **CSS** – Styling, layout, and animations
- **JavaScript** – Logic, interactivity, timers, and confetti animation
